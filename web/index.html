terminal-animations

A Claude Code plugin for authoring beautiful terminal animations — ASCII/ANSI motion for splash screens, loaders, demos, or just delight.

/plugin marketplace add ZviBaratz/terminal-animations
/plugin install terminal-animations@zvibaratz

The default answer to “make a cool terminal animation” is one effect, in flat 1×1 ASCII glyphs, with functional colour — competent and forgettable. A mesmerizing one is composed: a deliberate fidelity tier, a designed palette, layered effects, tuned by eye. The plugin exists to get the second thing, every time.
